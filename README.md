🌌 AirWriter Pro: Anti-Gravity Edition
A Futuristic Handwriting-to-Text Experience that Defies Gravity.

AirWriter Pro is an immersive Flutter application that allows users to write in a 3D spatial environment. Using advanced hand-tracking and a zero-gravity UI, it transforms simple gestures into flowing digital ink.

✨ Key Features
🛸 Zero-Gravity UI: Interactive floating elements with a premium glassmorphic design.

✍️ Spatial Writing: Write in the air! Your hand gestures are tracked in real-time to create seamless digital strokes.

✨ Neon Ink Flow: Smooth, glowing ink particles that follow your movement with realistic physics.

🌌 Ambient Space Environment: A dark, star-filled background designed for focus and aesthetic appeal.

📱 Spatial Menus: Orbiting controls that react to your presence.

🛠️ Tech Stack
Framework: Flutter (Web & Mobile)

State Management: Provider

Graphics: Custom Painter & CanvasKit Renderer

AI/ML: Google ML Kit & MediaPipe (for Gesture Tracking)

Styling: Glassmorphism & Neon Glow Effects

🚀 Getting Started
To get a local copy up and running, follow these simple steps:

Clone the Repository:

Bash
git clone https://github.com/muneeza-apps/Air-Writing.git
Install Dependencies:

Bash
flutter pub get
Run the Application:

Bash
flutter run -d chrome
📂 Project Structure
lib/main.dart: Entry point & App routing.

lib/widgets/: Futuristic UI components (Glass Canvas, Neon Loader).

lib/engine/: Logic for ink particles and gravity physics.

lib/providers/: State management for tracking coordinates.
